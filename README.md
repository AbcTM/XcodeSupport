Xcode真机支持组件.

### 使用方法

> 把此zip解压放到xcode的指定路径即可

#### Xcode路径

> Xcode.app -> 显示包内容 -> Contents -> Developer -> Platforms -> iPhoneOS.platform -> DeviceSupport

> 或者

	open ../../Applications/Xcode.app/Contents/Developer/Platforms/iPhoneOS.platform/Developer/



#### 下载zip
